# invest_companies

상장 기업별 **인터랙티브 시나리오 분석 앱** 모음. 각 기업 = `<ticker>/` 폴더의 정적 사이트(바닐라 JS, 빌드 없음).

- 라이브: https://jwj-nick.github.io/invest_companies/
- 생성 스킬: `equity-scenario-app` (자료조사→데이터→앱→배포)

## 수록
| 종목 | 코드 | 앱 |
|---|---|---|
| 에스에이엠티 (SAMT) | KOSDAQ 031330 | [samt/](https://jwj-nick.github.io/invest_companies/samt/) |

## 각 앱 구성
주가 추이 차트 · 연간/분기/배당/지역 시각화 · 사업구조·연혁 · 시나리오 슬라이더(배당·주가 4모델: PER/배당수익률/PBR/EV·EBITDA) · 달력형 이벤트 · 출처.

## 원칙
- 모든 수치에 출처 표기. 외부 출처가 아닌 추정치는 별도 표시.
- 산술 환산(예: 1분기×4)은 시나리오이지 전망이 아님.
- 원본 유료 리포트/PDF는 미수록(저작권). 공개 출처 링크로 연결.

> ⚠️ 본 자료는 투자 참고용이며 투자 권유가 아닙니다.
